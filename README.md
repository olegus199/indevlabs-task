# Indevlabs task

### Demo
You can view the demo by following [this link](https://olegus199.github.io/indevlabs-task/)

### Start
1. Clone
```bash
git clone https://github.com/olegus199/indevlabs-task.git
```
2. Go to the project dir
```bash
cd indevlabs-task
```
3. Install dependencies
```bash
pnpm i
```
or
```bash
npm i
```
4. Launch the project
```bash
pnpm run dev
```
or
```bash
npm run dev
```
5. Enter this in your borwser
```
http://localhost:5173/indevlabs-task
```
6. Run tests
```bash
pnpm run test
```
